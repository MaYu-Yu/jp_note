# 🎌 日語學習筆記本 (Japanese Learning Notebook)

這是一個專為日語學習者打造的全端 Web 應用程式。整合了 **單字 (Vocabulary)** 與 **文法 (Grammar)** 的筆記管理功能，並內建強大的 **單字卡 (Flashcards)** 系統，支援語音導讀 (TTS) 與自動播放，幫助使用者高效率複習日語。

---

## ✨ 核心功能

### 1. 智慧筆記管理系統
* **統一模板架構**：採用 `add_template.html` 統一管理新增與編輯邏輯，自動識別單字或文法模式。
* **Quill 富文本編輯器**：
    * 支援自定義字體、顏色、項目符號與編號清單。
    * **自動列表**：支援有序與無序清單，按下 Enter 自動換行並帶入格式。
    * **讀音標註系統**：透過自訂「讀」字按鈕，快速為單字添加 `[]` 標註。
    * **HTML 智慧辨識**：內建「🪄 辨識 HTML」功能，可直接貼上標準 HTML 清單代碼並自動渲染。
* **靈活分類**：支援多重標籤管理，新增時可動態建立新分類。

### 2. 智慧單字卡學習 (Flashcards)
* **客製化牌組**：可依「詞性」、「分類」或「資料類型」建立專屬學習範圍。
* **自動播放與語音導讀**：支援 3～10 秒自動翻頁循環，適合通勤或背景練習，內建語音朗讀功能。

---

## 🛠️ 編輯器使用說明

* **製作列表**：點擊工具列的 `bullet` (項目符號) 或 `ordered` (數字編號)，按下 `Enter` 鍵系統將自動延續格式。
* **標註讀音**：選取文字後點擊工具列上的「**讀**」按鈕，即可將文字包裝為 `[文字]` 格式。
* **貼上 HTML 代碼**：
    1. 將整理好的 `<ul><li>...</li></ul>` 代碼貼入編輯器。
    2. 點擊工具列上的「**🪄**」按鈕，編輯器會自動將其渲染為正式的列表格式。
* **清除格式**：若貼入的文字格式混亂，請使用「**Tx (Clean)**」按鈕一鍵恢復預設字體大小。

## 📁 專案架構概覽

* `app.py`: Flask 後端主程式，處理資料庫 CRUD 與 Session 管理。
* `templates/`:
    * `add_template.html`: 核心編輯器頁面模板（新增/編輯共用）。
    * `list_template.html`: 資料清單瀏覽與刪除管理。
    * `flashcard_deck.html`: 單字卡學習介面。
    * `categories_overview.html`: 分類標籤總覽與刪除管理。
* `instance/`: 存放 `jp_db.db` SQLite 資料庫檔案。

---

## ⚠️ 資料維護建議

* **ID 管理**：本專案採用 SQLite 自動遞增 ID。建議保留 ID 的連續性，若有刪除資料產生空缺，無需特別填補，資料庫效能不會受到影響。
* **資料清理**：若需將舊有的純文字 `・` 格式升級為 HTML 列表，建議執行自動化清理腳本，確保標籤結構（`<ul><li>`）乾淨且不含多餘換行。
* **資料庫瘦身**：若刪除大量資料後檔案大小未明顯縮減，可執行 `VACUUM;` 指令進行空間重組。
